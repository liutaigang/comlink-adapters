## 使用方法
- 点击 activitybar 上的图标： ![](https://raw.githubusercontent.com/liutaigang/vscode-webview-extension-example/main/documents/assets/activitybar-icon.png)
- 键入 `ctrl+shift+p` 在指令输入框中输入：panel-view-container.show
